O que é HTML5?

HTML5 é a quinta versão da Linguagem de Marcação de Hipertexto (HTML).

Ele foi lançado em 2014 e trouxe diversas novidades em relação à versão anterior:

  • Novas tags semânticas;
  • Novos atributos;
  • Novas APIs;
  • Maior compatibilidade.

Para muitos editores a criação de uma nova página em HTML5 já é automática, basta criar um novo documento baseado em um modelo e pronto.
